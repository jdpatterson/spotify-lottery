The Great Spotify Ticket Lottery Probability Calculator

This web app solves the Ticket Lottery problem implemented as a client-side application that can either calculate the answer in the browser or from a backend server running "in the cloud" (Google App Engine). The exact same Java source code is used in both cases - compiled to Javascript for the browser and to bytecode for the server.

  • Command line source reads from stdin.
  • StandardLotteryCalulator.java implements the business logic
  • LotteryCalulatorTestCase.java tests the sample inputs
  • Executable jar accepts parameters from standard in
  • The coder John Patterson on Facebook. Me!

The result is calculated when valid values are entered for each parameter
